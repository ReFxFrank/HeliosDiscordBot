# Birthdays

Celebrate member birthdays automatically.

## Settings

Dashboard → **Birthdays**: the announcement channel, the message, and an optional **birthday role** applied for the day. Announcements respect your server's timezone.

## Commands

| Command | What it does |
| --- | --- |
| `/birthday set month day` | Set your birthday |
| `/birthday remove` | Remove it |
| `/birthday list` | Upcoming birthdays this month |
