# Autoroles

Automatically apply roles the moment a member joins.

## Settings

Dashboard → **Autoroles**:

| Setting | What it does |
| --- | --- |
| Human roles | Applied to people who join |
| Bot roles | Applied to bots that join |

> Solari's role must sit **above** every autorole, with Manage Roles. If you run [Verification](/docs/verification) with an unverified gate role, keep autoroles compatible — the verified role comes from Verification, not here.
