# Command Toggles

Turn **individual slash commands** on or off for your server.

## Using it

Dashboard → **Slash Commands**. Every command Solari registers is listed live from Discord with a search box and a switch per command. Flip one off and it refuses everyone on your server — moderators included — within about a second, with a clear "disabled on this server" reply.

Common uses:

- Disable `/rob` while keeping the rest of Economy.
- Turn off casino games you don't want (or use the per-game switches on the [Casino](/docs/casino) page).
- Silence `/afk` or meme commands on a professional server.

Every toggle is recorded in the **Audit log**.

> Module switches (Overview grid) turn off a whole feature area; command toggles are the scalpel for single commands.
