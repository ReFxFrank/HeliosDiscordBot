# Music

Queue-based music playback in voice channels. A Premium module.

## Commands

| Command | What it does |
| --- | --- |
| `/play query` | Play or queue a track |
| `/pause`, `/resume`, `/skip`, `/stop` | Playback controls |
| `/queue`, `/nowplaying` | What's queued / playing |
| `/volume`, `/shuffle`, `/loop` | Volume, shuffle, and loop modes |

## Settings

Dashboard → **Music**: DJ roles and playback behavior.

> Music requires the Lavalink audio server on the bot host (bundled in the self-host stack via the `music` compose profile).
