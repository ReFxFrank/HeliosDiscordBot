# Logging

Server activity logs, split by category so each can go to its own channel.

## Setup

Dashboard → **Logging**, pick a channel per category:

| Category | Covers |
| --- | --- |
| Messages | Edits and deletions (single and bulk) |
| Members | Joins, leaves, bans and unbans |
| Server | Channel and role create/delete, server changes |
| Voice | Joins, moves, and leaves |

Leave a category blank to skip it.

> Tip: a single `#mod-logs` for everything works, but busy servers usually split messages into their own channel — edits and deletions are by far the noisiest category.
