# Custom Commands

Your own commands and auto-responses, with a full embed builder.

## Tags

Create named responses on the dashboard and members call them with `/tag name`. A tag's response can be plain text or a rich **embed** built visually (title, description, colors, fields, images — with a live Discord-style preview).

## Auto-responders

Trigger a response automatically when a message matches your trigger text — great for FAQ answers ("how do I verify?") and inside jokes.

## Commands

| Command | What it does |
| --- | --- |
| `/tag name` | Post a tag's response |

> The number of custom commands is limited on the free tier and raised with [Premium](/docs/premium).
