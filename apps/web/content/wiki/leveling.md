# Leveling & XP

Members earn XP for chatting and being active in voice; levels bring rank cards, role rewards, and a leaderboard.

## Settings

Dashboard → **Leveling / XP**:

| Section | Settings |
| --- | --- |
| XP Rates | XP per message (min–max), message cooldown, voice XP per minute |
| Level-Up Announcements | Where to announce (same channel / a set channel / DM / off) + the message |
| Rank Card & Leaderboard | Toggle the generated /rank card and the public leaderboard page |
| XP Restrictions | Roles and channels that earn no XP |
| Level Role Rewards | Grant roles at levels, stacking or replacing lower rewards |

## Public leaderboard

Every server gets a shareable, no-login leaderboard page at `solari.gg/leaderboard/<server id>` — real usernames, avatars, levels, and progress bars. Turn it off in **Rank Card & Leaderboard** if you'd rather keep it private.

## Commands

| Command | What it does |
| --- | --- |
| `/rank [user]` | Your (or someone's) level, XP, and rank — as a generated card |
| `/leaderboard` | The server's top members |

> Level-up messages support `{user}`, `{level}`, and `{server}` variables with a live preview in the editor.
