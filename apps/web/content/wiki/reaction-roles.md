# Reaction Roles

Self-assignable roles via **buttons** or a **select menu**, built visually on the dashboard.

## Building a panel

Dashboard → **Reaction Roles** → create a panel:

1. Give it a title and description (full embed editor with live preview).
2. Add options — each maps a button/menu entry (label + optional emoji) to a role.
3. Pick the style: **Buttons** or **Select menu**.
4. Pick the mode:

| Mode | Behavior |
| --- | --- |
| Normal | Each role toggles on/off independently |
| Unique | Picking one removes the others from the panel |
| Verify | Grant-only; clicking never removes |

5. Choose a channel and **Deploy**. Re-deploying updates the existing message in place.

> Solari's role must be above every role a panel hands out.
