# Starboard

Community-curated highlights: when a message collects enough ⭐ reactions, Solari re-posts it to your starboard channel with a link back.

## Settings

Dashboard → **Starboard**:

| Setting | What it does |
| --- | --- |
| Starboard channel | Where highlighted messages are posted |
| Threshold | Stars required before a message is boarded |

The starboard entry keeps its star count updated as reactions change.
