# Getting Started

Get Solari running on your server in a few minutes.

## 1. Invite the bot

Click **Add to Discord** on [solari.gg](/) and pick your server. You need the **Manage Server** permission on that server to add bots and to use the dashboard.

> Solari asks for Administrator by default so every module works out of the box. You can trim permissions later — each module page notes what it needs.

## 2. Sign in to the dashboard

Go to [solari.gg](/) and press **Login**. You'll authorize with Discord (we only read which servers you can manage — see the [Privacy Policy](/privacy)). Pick your server from the list.

## 3. Turn on your first modules

The server **Overview** shows every module as a card, grouped MEE6-style (Essentials, Server Management, Utilities, Social Alerts, Games & Fun). Flip a module's switch to enable it, then press **Configure** to open its settings page.

A good starter set:

1. **Moderation** — pick your mod/admin roles and a mod-log channel.
2. **Welcome & Goodbye** — greet new members.
3. **Leveling** — give your community XP to chase.
4. **Verification** — gate new members behind a button or captcha.

## 4. Check the basics in Settings

**Settings** (top of the sidebar) holds server-wide options: bot master roles, language, timezone, and the legacy text-command prefix.

## What next?

- Browse the module guides in the sidebar — each one covers setup, settings, and commands.
- See everything the bot can do in the [command reference](/commands).
- Curious about the paid tier? See [Solari Premium](/docs/premium).
