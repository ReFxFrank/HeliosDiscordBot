# Giveaways

Timed giveaways with entry buttons, role requirements, and rerolls.

## Running a giveaway

Use `/giveaway` (or the dashboard) to start one: prize, duration, number of winners, and optional **required roles** to enter. Members enter with one click; winners draw automatically when time's up.

| Command | What it does |
| --- | --- |
| `/giveaway` | Start and manage giveaways (end early, reroll winners) |

Giveaways are durable — they survive restarts and end on time. Entries also count toward the **Enter giveaways** achievement.
