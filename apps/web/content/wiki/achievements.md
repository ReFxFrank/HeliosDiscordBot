# Achievements

MEE6-style achievements members can hunt — **single** one-shot milestones or **tiered** ones (Bronze → Silver → Gold → Diamond) unlocked progressively.

## Tracked stats

| Type | Counts |
| --- | --- |
| Reach level | Leveling level |
| Send messages | Messages sent |
| Hold coins | Wallet + bank (Economy) |
| Minutes in voice | Voice activity |
| Add reactions | Reactions added |
| Create / join threads | Thread activity |
| Invite members | Attributed invites |
| Enter giveaways | Giveaway entries |
| Buy shop items | Economy purchases |
| Boost the server | Server boosts |
| Set a birthday | `/birthday set` |

## Building achievements

Dashboard → **Achievements**. Create a **Single achievement** (one threshold) or a **Tiered achievement** (up to four thresholds, Bronze → Diamond). Every tier can carry its own rewards: a role, coins, and XP. Use **Add starter set** for a curated pack (King of Spam, Stay Awhile and Listen, Reaction Master, and more).

Unlocks can be announced in a channel of your choice — tiered unlocks name the tier.

## Commands

| Command | What it does |
| --- | --- |
| `/achievements [user]` | Progress badges for every achievement |
| `/achievements-reset` | Admin: reset unlocks for a member and/or an achievement |
